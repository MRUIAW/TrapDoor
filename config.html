# Commands

每个命令都有enable和permissionLevel两个选项，前者控制改指令是否启用；后者控制指令的权限等级，设为1就是仅op可用，设置为0所有玩家可用，如果你要用到生存服中，请谨慎考虑权限和是否开启

# lowLevelVanillaCommands

这个配置能把仅后台可执行的指令降级为游戏内OP可执行,在内部填上相应的指令即可

# Server

该字段目前仅有levelName这一个选项，里面填写存档名称,和server.properties相同即可，backup指令需要该配置

# Village

配置村庄的显示效果,bound,spawn和poiQuery都只能选填 white,blue,red,green,yellow,最后一个center可以填任意合法粒子，包括其它材质包的自定义粒子,自定义粒子请加上名称空间。
